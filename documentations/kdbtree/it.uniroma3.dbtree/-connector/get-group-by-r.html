getGroupByR

abstract fun getGroupByR(y1: Any, y2: Any): List<NodeTransfer> (source)

Retrieve from the database all nodes n s.t. y1 ≤ n.k_min.y < y2 ∧ (y2 < n.k_max.y ∨ n.k_min.x ≠ n.k_max.x), ordered by ascending level.

This method can only be invoked when the key type is composite, i.e. admits a group and a least significant part.

Parameters

y1 - the minimum key in the range.

y2 - the maximum key in the range.

Return
the list of retrieved nodes.
