gtDB

abstract fun gtDB(columnName: String, k: K): String (source)

DB query for 'greater than' relationship.

Returns the string querying underlying DB for all elements having key (attribute) greater than a given value. Underlying dataset must know how to interpret the query, and return the expected result.

Parameters

columnName - the name of the database column interested in the comparison; since this method is only invoked from connector's queries, this field must refer to some columnName from this.keyColumnsDefinitionDB(); Additional checks over this field could be implemented to prevent runtime errors.

k - the key soil value.

Return
the string to be interpreted by underlying database.
